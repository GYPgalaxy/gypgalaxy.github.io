太极拳动作检测以及判断标准程度

python pyqt mysql pytorch

  • 项目描述：
    一个完整的太极拳动作检测系统，包括前后端。功能是通过一个单目摄像头和服务器，可以实现太极拳动作的实时检测，并且判断动作的标准程度。

  • 工作内容：

    • 基于Openpose框架通过摄像头获取单目标人物动作姿态，判定其太极拳动作分类。
    • 根据标准动作通过余弦运算语音提出动作矫正意见。
    • 基于PYQT开发应用程序，实现单目摄像头的信息获取以及后端模型计算的可视化反馈。
    • 指导学生撰写开发手册以及竞赛论文。

  • 成果：学生凭借该项目入围发明者大会中学组全国赛
